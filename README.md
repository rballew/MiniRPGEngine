\# Mini RPG Engine



A C++ console-based mini RPG engine where the player can explore, collect items, check stats, rest, and manage inventory.



\## Features



\- Player name creation

\- Player stats: health, attack, and gold

\- Inventory system using vectors

\- Exploration menu

\- Healing herb item collection

\- Resting system

\- Console-based menu loop

\- C++

\- Visual Studio

\- Git

\- GitHub
